# claude-verificador

> 🏗️ En construcción — el contrato VIGENTE está en `docs/FICHA.md`; el plan de obra, en el tracker (`docs/audits/`) y los issues #1–#4.

Convierte criterios de aceptación en tests trazables: cada criterio queda clavado por un esqueleto con nombre test↔criterio, o registrado como verificación manual explícita. Jamás un verde falso.

## Instalación

```bash
claude plugin marketplace add hifede1/claude-audit-tracker
claude plugin install verificador@fede-tools
```

Comando: `/verificador:criterios-a-tests` (v0.1.0 — esqueleto: declara el contrato, aún no genera tests; ver el plan S02–S04).
